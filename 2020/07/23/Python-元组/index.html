「元组」定义语法为：(元素1, 元素2, ..., 元素n)

  • 小括号把所有元素绑在一起
  • 逗号将每个元素一一分开

内置方法

元组只有两种内置方法：

  • count(x) 是记录在元组 t 中该元素出现几次
  • index(x) 是找到该元素在元组 t 的索引
